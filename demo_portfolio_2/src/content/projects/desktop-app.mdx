# Pro Task Manager

Masaüstü kullanıcıları için geliştirilmiş, sistem kaynaklarını minimum düzeyde kullanan bir görev yöneticisi.

## Özellikler

- **Cross-Platform**: Tek kod tabanı ile hem Windows hem de Mac üzerinde native deneyim.
- **Offline Sync**: İnternet bağlantısı koptuğunda yerel veritabanına yazma, bağlantı geldiğinde senkronizasyon.

### Teknik Detaylar

Uygulama, **Electron** ve **React** kullanılarak geliştirildi. Dosya sistemi erişimi için Node.js API'leri kullanıldı.
