<h2>
  <i18n lang="en" data-i18n="unity-choose-title">
    When to Choose Which?
  </i18n>
  <i18n lang="tr" hidden data-i18n="unity-choose-title">
    Ne Zaman Hangisini Seçmeli?
  </i18n>
</h2>

<i18n lang="en" data-i18n="unity-alert-unity" data-block>
  <Alert type="success" title="Choose Unity if...">
    - You're making a 2D game (Platformer, Top-down shooter)
    - You're targeting mobile platforms (iOS/Android)
    - You know C# or want to learn coding quickly
  </Alert>
</i18n>
<i18n lang="tr" hidden data-i18n="unity-alert-unity" data-block>
  <Alert type="success" title="Unity Seçin Eğer...">
    - 2D bir oyun yapıyorsanız (Platformer, Top-down shooter)
    - Mobil platformları hedefliyorsanız (iOS/Android)
    - C# biliyorsanız veya hızlıca kodlama öğrenmek istiyorsanız
  </Alert>
</i18n>

<i18n lang="en" data-i18n="unity-alert-unreal" data-block>
  <Alert type="warning" title="Choose Unreal if...">
    - You're aiming for AAA-quality visuals
    - You're building a 3D action game (FPS or TPS)
    - You want to build game logic without code (Blueprints)
  </Alert>
</i18n>
<i18n lang="tr" hidden data-i18n="unity-alert-unreal" data-block>
  <Alert type="warning" title="Unreal Seçin Eğer...">
    - AAA kalitesinde grafikler hedefliyorsanız
    - FPS veya TPS türünde 3D aksiyon oyunu yapıyorsanız
    - Kod yazmadan (Blueprints) oyun mantığı kurmak istiyorsanız
  </Alert>
</i18n>
